Equal-Sized Items with Flex-Shrink

This example creates a row layout where all items share the available space equally using flex: 1;. Additionally, flex-shrink: 1; is applied to allow all items to shrink proportionally if the container size is reduced and there's not enough space for their initial size. This ensures no item overflows the container.
